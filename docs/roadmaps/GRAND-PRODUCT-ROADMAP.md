# Grand product roadmap

**Themes**

1. **Trust & security** — Auth (Clerk), tenancy, auditability, data residency options.
2. **Knowledge quality** — Ingestion reliability, source attribution, evaluation of RAG answers.
3. **Scale** — Registry on Postgres, multi-region, worker pools for ingestion.
4. **Adoption** — Web UX, admin tooling, demos per vertical.

Phases below split work into deliverable slices; adjust dates with product.
