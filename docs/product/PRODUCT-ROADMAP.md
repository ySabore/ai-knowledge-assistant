# Product roadmap (high level)

1. **Hardening** — Clerk JWT auth, rate limits, audit logging for chat/ingest.
2. **UX** — Workspace picker, chat UI, ingestion status in the web app.
3. **Enterprise** — Postgres registry, SSO, RBAC roles beyond membership.
4. **Connectors** — Drive, SharePoint, S3 ingestion (policy-gated).
